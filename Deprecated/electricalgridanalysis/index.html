Project Overview: Electrical Grid Analysis

    Developed as a final project for CMPT 318: Cybersecurity, Simon Fraser University.

    Collaborated in a team of four to apply data analysis methodologies, such as simple point anomaly detection and markov model training.

Analysis was performed on sample US electrical power grid data sets ranging from 2007-2011. Represented by a 'clean' data set, and a 'anomalous' data set which were used in conjuction to determine training benchmarks.

RStudio packages used:

  • 'ggplot2' for graphing and simple point anomaly detection
  • 'TTR' for statistical functions such as moving averages
  • 'depmixs4' for hidden markov model training and algorithms.

Documentation: Presentation

A 'fun' powerpoint presentation, summarizing the findings of our analysis and showcasing the methodologies for anomaly detection.
